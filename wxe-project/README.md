# wxe-project
WXE Project:
wxe-Linux
Нативная поддержка Windows-программ на Linux — без эмуляции, без Wine.
Что это:
Кастомное ядро + модуль совместимости с Windows API. Всё работает нативно, быстрее и стабильнее.
Зачем:
Игры и софт Windows на Linux без костылей
Максимальная производительность
Новая эра совместимости
